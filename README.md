# 💫 About Me:
I’m a Computer Science Engineering student who focuses on building real, working systems instead of just collecting buzzwords. I work across backend development, data science, and core computer science fundamentals, with hands-on experience in Python, Java, SQL, and web technologies. I’ve built projects involving APIs, authentication, data analysis, machine learning pipelines, and real-time monitoring systems, and I care deeply about understanding how things work under the hood rather than just making things look good.
